2. How to use
- 명령 프롬프트에서 파이썬을 실행하고 BST를 import 하면 사용가능합니다.

- 기본적으로 5가지 함수가 존재합니다.
1. insert(value)
2. delete(value)
3. print_preorder()
4. print_inorder()
5. print_postorder()
